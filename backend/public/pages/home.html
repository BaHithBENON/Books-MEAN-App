Documentation de l'API

Introduction
Bienvenue dans la documentation de l'API de gestion de livres. Cette API fournit des fonctionnalités pour gérer les données de livres, y compris l'ajout, la mise à jour, la suppression, et la récupération d'informations spécifiques sur les livres.
Points d'extrémité (Endpoints)

Url Base : https://books-mean-app.onrender.com

Récupérer toutes les données.

  • Endpoint : /books
  • Méthode : GET
  • Description : Récupère toutes les données des livres.

Ajouter une nouvelle donnée.

  • Endpoint : /books
  • Méthode : POST
  • Description : Ajoute une nouvelle donnée de livre.
  • Paramètres requis : Les données du livre doivent être incluses dans le corps de la requête.

Récupérer une donnée spécifique.

  • Endpoint : /books/:id
  • Méthode : GET
  • Description : Récupère une donnée de livre spécifique en utilisant l'identifiant.

Supprimer une donnée.

  • Endpoint : /books/:id
  • Méthode : DELETE
  • Description : Supprime une donnée de livre en utilisant l'identifiant.

Mettre à jour une donnée existante.

  • Endpoint : /books/:id
  • Méthode : PUT
  • Description : Met à jour une donnée de livre existante en utilisant l'identifiant.
  • Paramètres requis : Les nouvelles données du livre doivent être incluses dans le corps de la requête.

Récupérer les livres par catégorie.

  • Endpoint : /books/category/:category
  • Méthode : GET
  • Description : Récupère les livres appartenant à une catégorie spécifique.

Récupérer les livres par année.

  • Endpoint : /books/year/:year
  • Méthode : GET
  • Description : Récupère les livres publiés une année spécifique.

Récupérer les livres par note moyenne.

  • Endpoint : /books/rating
  • Méthode : GET
  • Description : Récupère les livres triés par note moyenne décroissante.

Pagination des résultats.

  • Endpoint : /books/paginated
  • Méthode : GET
  • Description : Récupère les livres paginés en fonction du numéro de page et de la taille de la page.

Moyenne des notes de tous les livres.

  • Endpoint : /books/average-rating
  • Méthode : GET
  • Description : Récupère la moyenne des notes de tous les livres.

Filtrage par nombre de pages.

  • Endpoint : /books/pages
  • Méthode : GET
  • Description : Récupère les livres avec un nombre de pages supérieur ou égal à la valeur spécifiée.

Statistiques générales.

  • Endpoint : /books/statistics
  • Méthode : GET
  • Description : Récupère des statistiques générales, telles que le nombre total de livres et la note moyenne.

Recherche texte intégral.

  • Endpoint : /books/search
  • Méthode : GET
  • Description : Effectue une recherche texte intégral basée sur un mot-clé.

Récupérer toutes les catégories de livres.

  • Endpoint : /books/categories
  • Méthode : GET
  • Description : Récupère toutes les catégories de livres disponibles.

Récupérer tous les auteurs de livres.

  • Endpoint : /books/authors
  • Méthode : GET
  • Description : Récupère tous les auteurs de livres disponibles.

Récupérer toutes les années de publications de livres.

  • Endpoint : /books/years
  • Méthode : GET
  • Description : Récupère toutes les années de publications de livres disponibles.
